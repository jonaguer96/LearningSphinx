What is restructuredText?

reStructuredText is an easy-to-read, what-you-see-is-what-you-get plaintext markup syntax and parser system. It is useful for in-line program documentation (such as Python docstrings), for quickly creating simple web pages, and for standalone documents. All throughout the course, we will be covering what one needs to install to write documentation with restructured text formatting with Sphinx via VS Code.
